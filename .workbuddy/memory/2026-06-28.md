- **构建验证**: TypeScript ✔ | Vite ✔ | Cargo ✔
- **发布**: 已推送到 GitHub (v1.17.1)
- **注意**: `git push` 在 bash 中因 credential manager 不可用会失败，需用 `GIT_ASKPASS=echo git push` 解决

## v1.17.2 — 修复日历视图深色模式样式

- **问题1**: 月/周/日切换按钮深色模式下未选中按钮文字不可见（`text-gray-500` 在深色背景消失）
- **问题2**: 日历背景色与深色系统不协调（大块 `bg-white`/`bg-[#FAFAFA]` 未适配深色）
- **修复**:
  - CalendarView.tsx: 工具栏、ViewToggle、MoreOptions、TaskSidebar
  - MonthView.tsx: 日历导航栏、日期格子、新建任务弹窗
  - WeekView.tsx: 完整深色适配（导航、时间轴、弹窗）
  - DayView.tsx: 完整深色适配
- **核心策略**: `bg-white`→`dark:bg-gray-900`, `border-gray-200`→`dark:border-gray-700`, `text-gray-500`→`dark:text-gray-400`, `hover:bg-gray-100`→`dark:hover:bg-gray-700`
- **构建验证**: TypeScript ✔ | Vite ✔
- **发布**: 已推送到 GitHub (v1.17.2)
- **注意**: tauri.conf.json 之前滞后于 v1.17.1，本次一并同步到 1.17.2
